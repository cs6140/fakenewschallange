# fakenewschallange

Analysis of the performance of ML  algorithms on the classification of Fake News into 4 discrete levels

## Script Dependencies - Python modules

* Feature Generation
   	   
1) nltk
2) nltk-stopwords
3) nltk-wordnet
4) Gensim (word2Vec)	


* Exploratory Data Analysis and Visualization
    	    	 	
5) pandas
6) seaborn 

   	  
* Algorithms

7) Scikit Learn
8) XGBoost
